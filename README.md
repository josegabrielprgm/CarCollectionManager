# CarCollectionManager

Sistema desktop para gerenciamento de colecionadores e veículos.

## Tecnologias
- Java
- Swing
- JPA (Hibernate)
- MySQL

## Funcionalidades
- Cadastro de colecionadores
- Cadastro de carros
- Alterar e excluir registros

## Como executar
1. Clonar o projeto
2. Configurar o banco de dados
3. Executar a classe principal
